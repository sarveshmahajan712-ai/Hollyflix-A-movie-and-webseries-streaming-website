# 🎬 Hollyflix

Hollyflix is a responsive movie streaming website UI built using HTML, CSS, and Bootstrap. It is inspired by modern OTT platforms and focuses on creating an attractive and user-friendly interface.

## 🚀 Features

* 🎥 Hero section with Bootstrap carousel
* 📂 Movie sections using grid layout
* 🎞️ Scrolling content using marquee
* 🎴 Movie cards with hover animations
* 📝 Contact / Signup form
* 📱 Fully responsive design
* 🔻 Stylish footer section
* ✨ Smooth UI animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## 🎯 Concepts Used

* Bootstrap Grid system
* Carousel (Slider)
* Marquee effect
* Cards layout
* Forms (Input fields, buttons)
* Animations & transitions
* Responsive design

## 📝 Form Features

* User input fields (Name, Email, etc.)
* Styled using Bootstrap
* Responsive layout
* Basic validation (if applied)

## 📌 Purpose

This project was created to practice frontend development skills and understand layout design using Bootstrap components.
